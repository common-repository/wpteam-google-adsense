WPteam present a simple Google Adsense that easy to copy-paste Adsense's code to Wordpress post.

== Description ==

For easy copy-paste Google Adsense's code to Wordpress's Site, We decided to build a plugin that provide simple header's script and fixed position for the post.
This plugin is not just a simple and useful for Google Adsense's copy-paste but also provide the space for put your own banner script on the Wordpress's site.
Include some simple banner on WooCommerce Cart and Checkout page.

We hope that our plugin will make Wordpress's Site more simple and ready for everyone who want to start their new base.

== Installation ==

Simple download plugin and install for Wordpress Plugin directory.

== Frequently Asked Questions ==

= My Google Adsense Account did not active, Help! =

We wish we can do something to help but we can not. If you did not had an active Google Adsense Account. It not necessary to installed this plugin.
